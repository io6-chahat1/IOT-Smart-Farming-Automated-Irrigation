# IoT Smart Farming and Automated Irrigation System

## Description
This project is an IoT based automated irrigation system that monitors soil moisture, temperature and humidity in real time and automatically controls a water pump.

## Hardware Used
- NodeMCU ESP8266
- Soil Moisture Sensor
- DHT22 Sensor
- Relay Module
- Water Pump

## Software Used
- Arduino IDE
- Blynk App
- GitHub

## Working
The system collects soil moisture, temperature and humidity data.
If soil moisture is low, the water pump turns ON automatically.
If moisture is sufficient, the pump turns OFF.

## Features
- Automatic irrigation
- Real-time monitoring
- Water conservation
- Mobile app control
